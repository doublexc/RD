# 🚀 RD - Simple URL Shortener
ระบบย่อลิงก์ส่วนตัวที่ใช้ GitHub Pages ร่วมกับ Google Sheets

### 📌 วิธีใช้งาน
1. เพิ่ม ID และ URL ปลายทางใน Google Sheets
2. เข้าใช้งานผ่านลิงก์: `https://doublexc.github.io/RD/?id=YOUR_ID`
3. (Optional) ใช้งานผ่าน TinyURL: `https://tinyurl.com/paitee?id=YOUR_ID`

### 🛠️ เทคโนโลยีที่ใช้
- **GitHub Pages:** สำหรับ Hosting หน้าเว็บ Static
- **Google Sheets:** สำหรับเป็นฐานข้อมูล (Database) ผ่านรูปแบบ CSV
- **JavaScript (Fetch API):** สำหรับประมวลผลการ Redirect
